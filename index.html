Datos Generales de la Materia

Descripción del curso: Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web. A través del dominio de los elementos del lenguaje HTML y el uso de los elementos de estilo en lenguaje CSS los estudiantes podrán diseñar sitios web atractivos y funcionales. Mediante el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podrán crear aplicaciones interactivas de front-end e introducir contenido dinámico. Asimismo, serán capaces de comprender la arquitectura de las aplicaciones front-end, el rol de cada tecnología en la creación de aplicaciones modulares y conocerán las mejores prácticas en el diseño de estas. Finalmente, se discutirán algunas prácticas desventajosas en el desarrollo de software.

Contenidos:

Unidad 1: Introducción al HTML
Elementos HTML. Semántica del HTML. Atributos HTML.

Unidad 2: Introducción a las hojas de estilo en cascada (CSS)
Selectores CSS. Propiedades CSS. Modelo de caja.

Unidad 3: Maquetación de sitios web
Posicionamiento y visualización. Diseño de layout básico. Diseño responsivo y media queries. Introducción a Flexbox.

Unidad 4: Frameworks de diseño web responsivo
Malla de Bootstrap. Utilidades para Bootstrap. Elementos Bootstrap.

Unidad 5: Introducción a Javascript
Introducción a Javascript y su sintaxis. Estructuras de control. Arreglos y objetos. Funciones y eventos. Programación Orientada a Objetos en Javascript.

Unidad 6: El Modelo de Objetos de Documento (DOM) y Javascript asíncrono
Manipulación del Document Object Model (DOM). Notación de Objetos de Javascript (JSON). Almacenamiento local: localstorage y sessionstorage. Programación asíncrona en Javascript. AJAX. Consumo de APIs. Almacenamiento de datos en Firebase.
